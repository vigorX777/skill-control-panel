# Skill Control Panel V2 功能清单

## 1. 当前基线

- 版本：`V2 / current-fact governance`
- 更新日期：`2026-07-12`
- 原则：Registry 只记录当前事实；浏览器只读；所有变更经严格 `skillctl` 契约执行。

状态含义：`已实现` 表示已有实现和自动化测试；`已移除` 表示 V1 能力不属于 V2 对外契约。

服务无通配 CORS；并发首次读取共享同一个扫描 Promise；显式 refresh 在当前扫描完成后强制执行新扫描。HTTP 请求不调用 manager 写方法。
